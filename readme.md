# Slim3 boilerplate

### Installation

Boilerplate requires [PHP](http://php.net/docs.php) >=v7.1 to run.

Install the dependencies and start the server.

```sh
$ composer install
```

### Cli-commands

Create new controller:
```sh
$ php slim controller:create %name %actions
```

Create new model:
```sh
$ php slim model:create %name %table
```
